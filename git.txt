Ayuda para github

Create a new repository on the command line
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/crios2020/java.utilities.git
git push -u origin main

Push an existing repository from the command line
git remote add origin https://github.com/crios2020/java.utilities.git
git branch -M main
git push -u origin main
